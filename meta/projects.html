Fronkin Geetah Chordz

https://github.com/FTLam11/fronkin_geetah_chordz

Fronkin Geetah Chords visualizes notes and intervals on a guitar fretboard for any guitar tuning. Input any six string tuning, key, chord root, and chord type to view a color coded set of notes and musical intervals.

Audio Station Scrobbler

https://github.com/FTLam11/Audio-Station-Scrobbler

Audio Station Scrobbler is a light-weight service for Synology NAS devices. It scrobbles music using the Last.fm API, as there is currently no support for Audio Station.

Fronkin Bandcamp Scraper

https://github.com/FTLam11/fronkin_bandcamp_scraper

Fronkin Bandcamp Scraper is a convenient Ruby gem that scrapes artist/release information from Bandcamp.
