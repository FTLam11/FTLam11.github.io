Things I learned doing this challenge:

  • Using Chrome's developer tools is really convenient. I used it to inspect any element of interest and view the CSS rules that applied to it. Being able to manipulate the CSS and observe the changes real-time was good for experimenting. Any rule that worked out could then be saved to a copy of the CSS file.
  • I got a small preview of next week's module on CSS. It'll be more interesting to style our sites.
  • The most troubling part was figuring out how to display the links inline on a mobile display. I was trying to target the ul tags with display: inline" when I should have been targeting the li tags and the #side-a div.
